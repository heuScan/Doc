## 项目定位

本项目定位为面向渗透测试与安全评估场景的智能信息收集全链路工具，通过统一采集入口与 Agent 辅助决策，提升信息收集效率、完整性与准确性，并为后续漏洞挖掘提供高质量资产基础。

## 核心能力概览

系统围绕“发现—解析—研判—输出”构建能力闭环：在资产发现与关联层面，覆盖公司主体、备案、子域名、IP、端口、路径及技术栈信息，并建立企业与资产关系图谱；在扫描与解析层面，支持基于目标特征的动态字典生成、JS 语义深度解析与外联线索提取；在流量与风险层面，支持 HTTP 被动审查、漏洞线索标记与高风险目标二次审计；在结果输出层面，可生成网络拓扑、价值分级结果与信息收集报告，支撑后续渗透工作推进。部分能力（如云服务与代码仓库泄露识别、Agent 决策调度增强）作为扩展方向规划中。

## 项目亮点

- **动态扫描字典**：结合技术栈与业务特征动态扩展路径，减少无效扫描，提升命中率。  
- **闭环风险研判**：主动扫描与被动流量审查联动，对高风险目标进行二次审计与优先探测。  
- **目标价值排序**：对资产与线索进行价值标记，帮助快速识别高价值目标与潜在攻击链路。  
- **自动化成果输出**：统一沉淀关系图谱、网络拓扑与信息收集报告，降低人工整理成本。
